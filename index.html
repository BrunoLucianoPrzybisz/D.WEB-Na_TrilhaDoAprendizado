Sobre o HTML

Um pouco sobre HTML, o que é ? como funciona ?

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto" é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores. A tecnologia é fruto da junção entre os padrões HyTime e SGML. É uma linguagem de marcação de texto através das TAGs, que permite apresentar informações na internet; como links, imagens e o próprio texto. Todas as páginas que você abre são a leitura e a interpretação de um arquivo que contém esse código HTML.
Tipo de codigo: TEXT.
Inventor: Tim Berners-Lee.
Desenvolvido por: W3C & WHATWG.
Tipo de formato: Linguagem de marcação.

Imagem

TAG:< img >

No HTML, a tag responsável pela inserção de imagens é a tag < img >. Porém, o HTML tecnicamente não faz a inclusão da imagem em si, o que ele faz é linkar a imagem para a página, de forma que ela seja aberta como se estivesse inserida na mesma, parecido com o processo da tag link <a >

Para inserir uma imagem no HTML basta utilizar a tag < img > com o atributo src. Ou seja: o atributo src, ou source, vai conter a url da imagem que será inserida. Diante disso, a sintaxe final será:
<img src="url">
 Para inserir uma imagem local, podemos apenas incluir o nome da imagem com sua extensão, como no exemplo abaixo:
<img src="imagem.jpg"> Se ela estiver contida dentro de uma pasta local, nós devemos referenciar a pasta também. Por exemplo, se minha página principal está contida na pasta public_html e dentro dela temos uma pasta chamada imagens, onde está a nossa imagem, utilizaremos a seguinte url:
<img src="imagens/imagem.jpg">

Exemplo da implementação do HTML5:
<figure>
<img> src="imagem.jpg" alt="Minha Figura">
<figcaption>Informações da Figura</figcaption>
</figure>

Listas

Suporta listas ordenadas, não ordenadas e de detalhadas.

Sintaxe

Ordenadas: <ol>
<li> Café </li>
<li> Leite </li>
</ol>

Não Ordenadas: <ul>
<li> Café </li>
<li> Leite </li>
</ul>

Definições:

Ordenadas:
Uma lista ordenada é também uma lista de itens. As listas de itens são marcadas com números. Uma lista ordenada começa com a tag <ol>. Cada item da lista começa com a tag <li>
Aqui está como aparece em um navagador:


  1. Café

  2. Leite


Não Ordenadas:
Uma lista não ordenada é uma lista de itens. As listas de itens são marcadas com bullets (tipicamente pequenos círculos pretos).Uma lista não ordenada começa com a tag <ul>. Cada item da lista começa com a tag <li>
Aqui está como aparece em um navegador :


  • Café

  • Leite


Listas Detalhadas:
<dl>
<dt> Café </dt>
<dd> Descrição </dd>
<dt> Leite </li>
<dd> Descrição </dd>
</dl>
Aqui está como aparece em um navegador :


Café
Descrição
Leite
Descrição

Hyperlinks

O que é ? como funciona ?

Seu Objetivo:
Definição:

É uma referência dentro de um documento em hipertexto para outras partes do mesmo documento, outros documentos,arquivos ou endereço eletrônico. Os links podem se classificar em:


  1. remotos ou externos;
  2. internos;
  3. locais;
  4. com endereço de correio;
  5. arquivos.

Marcação de Hiperlinks:
acontece quando há a utilização da tag "<a>" e a propriedade "href" que faz a referência para o conteúdo desejado.

Sintaxe:

<a href="endereço">Conteúdo Referenciado Clicável /a>

TIPOS DE HIPERLINKS:

  • Hiperlinks externos ou remotos: São links que referenciam sites ou páginas da web. Na propriedade href utilizamos a URL(Uniform Resource Locator) para a referência que desejamos.
  • Sintaxe:<a href="URL">conteúdo referenciado./.a >
  • Hiperlinks internos: São links que apontam a um lugar diferente dentro da mesma página e possibilita acessar diferentes parágrafos ou seções quando o documento é extenso.
  • Sintaxe:<a href="#referencia">conteúdo referenciado. /.a> ; <a name="referencia><. /.a >
  • Hiperlinks locais: São links que apontam para outras páginas dentro de um mesmo site.
  • Sintaxe:<a href=" arquivo.html">conteúdo referenciado
  • Hiperlinks em imagens: É possível realizar hiperlinks imagens envolvendo a tag de uma imagem com a tag de hiperlink
  • Sintaxe:<a href="referencia">< img src="imagem.jpg"> . /.a>
  • Hiperlinks para downloads de arquivos: São links que apontam para arquivos e possibilitam realizar download destes arquivos.
  • Sintaxe:<a href="arquivo.extensão" download="arquivo.extensão" type="tipo">conteúdo<. /.a>
  • Hiperlinks para endereço eletrônico: São links que apontam para um endereço eletrônico.
  • Sintaxe:<a href="mailto:email@email.com">email@email.com.
